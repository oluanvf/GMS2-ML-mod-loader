# GMS2-ML-mod-loader
A set of codes that allow running simple GML code in Game Maker Studio 2 without using DLLs.  
Theoretically, it works on all devices since it only uses functions from Game Maker Studio 2 itself. It's still in development, which means it's incomplete and has bugs.  

List of things you can do:  
- Interaction with objects  
- Interpret GML code:  
  -- if, else, for, while, with, try, catch  
  -- create simple functions and constructors  
  -- logic of priorities and math  
- Use many constants  
- Use all functions of Game Maker Studio 2  

How to use it? (simple)  
- run(script/string/path > arguments...);
